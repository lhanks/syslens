# Syslens Brand Identity

## Name Evaluation

### Current Name: Syslens

**Verdict: RECOMMENDED - Keep "Syslens"**

**Strengths:**
- Short, memorable, and pronounceable (7 letters)
- Clearly conveys system monitoring/visibility ("Sys" + "lens")
- Technical but not overly complex
- Unique in the market - stands out from generic tools
- Domain availability: syslens.io, syslens.app, syslens.dev all available
- Strong verb/noun portability: "lens into your system", "view through Syslens"

**Domain Availability:**
- syslens.com: TAKEN (but not critical for desktop app)
- syslens.io: AVAILABLE (recommended)
- syslens.app: AVAILABLE (strong alternative)
- syslens.dev: AVAILABLE (developer-focused)
- syslens.tech: AVAILABLE

**Alternative Names Considered:**
1. SystemLens - More obvious but less memorable, longer
2. PulseView - More generic, many domains taken
3. SysVue - Vue.js framework confusion
4. InsightSys - Less clear purpose
5. MetricLens - Too focused on one feature

## Visual Identity Summary

### Logo Concept
The Syslens logo combines a circular lens element with system metrics visualization. The icon features:
- Circular lens frame representing focus and clarity
- Graph/chart elements showing system monitoring
- Gradient treatment suggesting live, dynamic data
- Modern, minimal geometric style

### Color Palette

**Primary Colors:**
- **Deep Space** (#0F172A) - Main background, professional depth
- **Electric Blue** (#3B82F6) - Primary brand color, interactive elements
- **Cyber Purple** (#8B5CF6) - Secondary accent, data visualization

**Data Visualization:**
- **Success Green** (#10B981) - Healthy metrics, positive states
- **Warning Amber** (#F59E0B) - Warning states, attention needed
- **Critical Red** (#EF4444) - Errors, critical alerts
- **Info Cyan** (#06B6D4) - Information, network metrics

**Neutral Scale:**
- **Slate 950** (#020617) - Darkest backgrounds
- **Slate 900** (#0F172A) - Main backgrounds
- **Slate 800** (#1E293B) - Cards, panels
- **Slate 700** (#334155) - Borders, dividers
- **Slate 400** (#94A3B8) - Secondary text
- **Slate 200** (#E2E8F0) - Primary text
- **White** (#FFFFFF) - Brightest text, data values

### Typography

**Display/Headings:**
- **Font:** Inter (system: -apple-system, BlinkMacSystemFont, "Segoe UI")
- **Weights:** 700 (Bold), 600 (Semibold)
- **Usage:** App title, section headers, metric labels

**Body/UI:**
- **Font:** Inter
- **Weights:** 400 (Regular), 500 (Medium)
- **Usage:** Navigation, descriptions, tooltips

**Data/Monospace:**
- **Font:** JetBrains Mono (fallback: "Consolas", "Monaco", monospace)
- **Weights:** 400 (Regular), 500 (Medium)
- **Usage:** Numbers, percentages, technical values, process IDs

## Usage Guidelines

### Logo Usage

**Do:**
- Use on dark backgrounds (primary)
- Maintain minimum clear space (equal to lens diameter)
- Scale proportionally
- Use provided color versions

**Don't:**
- Distort or stretch
- Add effects (drop shadows, glows)
- Use on busy backgrounds without backdrop
- Place text too close to logo

### Color Usage

**Backgrounds:**
- Primary: Deep Space (#0F172A)
- Secondary: Slate 950 (#020617)
- Cards/Panels: Slate 800 (#1E293B)

**Interactive Elements:**
- Primary Action: Electric Blue (#3B82F6)
- Secondary Action: Cyber Purple (#8B5CF6)
- Hover States: Lighter shade (+10% lightness)

**Status Indicators:**
- Success: Green (#10B981)
- Warning: Amber (#F59E0B)
- Error: Red (#EF4444)
- Info: Cyan (#06B6D4)
