# Plan: Task 2 - Add System Report Export Feature

## Objective

Add ability to export a complete system report to a file, allowing users to save all collected system information.

## Investigation Summary

**Available Data Sources:**
- **System**: device_info, bios_info, boot_config, os_info, uptime, domain_info, user_info
- **Hardware**: cpu_info, memory_info, gpu_info, motherboard_info, usb_devices, audio_devices, monitors
- **Storage**: physical_disks, volumes, network_drives
- **Network**: network_adapters, routing_table
- **Services**: services summary

**Technical Details:**
- Tauri 2.0 with `tauri-plugin-fs` already available
- File dialog available via Tauri's dialog API
- All data models use serde serialization
